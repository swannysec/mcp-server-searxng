This MCP server requires access to a SearXNG instance.

1. Choose a SearXNG instance:
   - Use a public instance from [searx.space](https://searx.space/)
   - Or [self-host your own](https://docs.searxng.org/admin/installation.html)

2. Configure the instance URL in your Zed settings (required)

**Optional:** Configure authentication, proxy settings, or custom User-Agent if needed.
